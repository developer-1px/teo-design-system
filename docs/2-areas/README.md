# 2. Areas (지속 관리 영역)

> **계속 유지하고 업데이트해야 하는 표준과 지식**

---

## 📋 개념

Areas는 다음과 같은 영역입니다:

- ✅ **데드라인 없음**: 지속적으로 관리
- ✅ **표준 유지**: 항상 최신 상태로 유지
- ✅ **책임 영역**: "이것은 내가 관리한다"

Projects와 달리, Areas는 **완료되지 않습니다**. 대신 **계속 업데이트**됩니다.
