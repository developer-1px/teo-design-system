## 📋 개념

Projects는 다음 조건을 만족하는 작업입니다:

- ✅ **명확한 목표**: "무엇을 완성할 것인가?"
- ✅ **데드라인**: "언제까지?"
- ✅ **완료 가능**: 완료되면 Areas 또는 Archive로 이동

## 📝 프로젝트 추가 방법

1. `1-projects/` 폴더에 프로젝트 이름으로 폴더 생성
2. 프로젝트 폴더 내 `README.md` 작성:
   - 목표
   - 데드라인
   - 진행 상황
   - 관련 문서 링크

3. 완료 시:
   - 지속 관리 필요 → `2-areas/`로 이동
   - 비활성화 → `4-archive/`로 이동
