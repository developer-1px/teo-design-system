---

## 아카이브 정책

다음 조건에 해당하는 문서는 이 폴더로 이동됩니다:

1. **Outdated** - 최신 버전으로 대체됨
2. **Deprecated** - 더 이상 사용하지 않는 기능/패턴
3. **Duplicate** - 중복 문서 (최신 버전 유지)
4. **Draft** - 임시/초안 문서 (정식 버전 존재)
5. **Personal** - 개인 메모/임시 작업 파일
