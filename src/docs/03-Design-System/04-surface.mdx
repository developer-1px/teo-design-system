# Surface System

`Surface`는 디자인 시스템의 기초가 되는 배경색과 계층(Elevation) 구조를 정의합니다.
본 시스템은 단순히 배경색을 지정하는 것이 아니라, **배경색(bg) + 테두리(border) + 그림자(shadow) + 텍스트 색상(text)**을 하나의 계약(Contract)으로 묶어 관리합니다.

## 핵심 개념

### 1. Base
최하단 레이어입니다. 애플리케이션의 전체 배경이나 메인 컨테이너의 기본 배경으로 활용됩니다.
- **주요 용도:** `App Background`, `Main Layout`
