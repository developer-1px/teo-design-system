# Typography System

우리 디자인 시스템의 타이포그래피는 **가독성(Readability)**과 **계층 구조(Hierarchy)**를 명확히 하는 것을 목표로 합니다.
`vars.css.ts`에 정의된 토큰들은 `fontSize`, `weight`, `lineHeight`의 조합으로 사용됩니다.

## Best Practices

### Headings
*   `h1` (Page Title): `fontSize.xxl` + `weight.bold`
*   `h2` (Section Title): `fontSize.xl` + `weight.bold`
*   `h3` (Subsection): `fontSize.lg` + `weight.bold`

### Body Text
*   기본 본문은 `fontSize.md` (13px) 또는 `fontSize.lg` (14px)를 사용합니다.
*   줄 간격은 `lineHeight.standard` (1.5)를 기본으로 합니다.

### Code
*   소스 코드는 `font.code` (Menlo, Monaco...)를 사용하며, 가독성을 위해 `fontSize.sm`을 권장합니다.
