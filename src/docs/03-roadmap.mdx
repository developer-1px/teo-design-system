# Design System Roadmap

현재 디자인 시스템의 구현 상태와 앞으로의 계획입니다.

## 1. 완료된 기능 (Implemented)

### Foundation (Tier 1)
- [x] **Surface System**
  - Base, Subtle, Card, Highlight, Ghost 계층 정의
  - Light/Dark 모드 완벽 지원
  - `vars.surface` 토큰화 완료
- [x] **Typography System**
  - Semantic Scale (Title, Subtitle, Body, Caption)
  - `vars.fontSize`, `vars.weight`, `vars.lineHeight` 토큰화
- [x] **Layer System** (ex. Overlay)
  - **Positioning Layers**: Sticky, Fixed, Absolute
  - **Semantic Overlays**: Modal, Drawer, Tooltip
  - Z-Index & Elevation 토큰화 완료
- [x] **DevTools**
  - Vite Plugin Inspector (Meta+D)
  - Source Code Jump 기능

## 2. 진행 중 (In Progress)

### Layout System (Tier 1)
- [ ] **Spacing & Rhythm**
  - `vars.spacing` 토큰 재정비 (4px 그리드 시스템 강화)
  - 모바일/데스크탑 반응형 간격 정의
- [ ] **Grid & Stack**
  - Layout Primitive 컴포넌트 (`<Box>`, `<Stack>`, `<Grid>`)
  - CSS Subgrid 활용 패턴

## 3. 예정됨 (Backlog)

### Motion System (Tier 2)
- [ ] **Transition Tokens**
  - `vars.duration`, `vars.easing` 정의
  - 감각적인 마이크로 인터랙션 (Hover, Press, Dialog Enter/Exit)

### Component Library
- [ ] **Form Elements**
  - Input, Select, Checkbox, Switch (제어 컴포넌트)
  - Validation State (Error, Success)
- [ ] **Data Display**
  - Table, List, Tag, Badge
