# Future Roadmap

`Common Patterns`에서 도출된 잔여 과제 및 향후 고도화 예정인 컴포넌트들입니다.

## 1. PageShell (Layout System)
도구 지향적 인터페이스의 고수준 레이아웃(Sidebar + Header + Content)을 관리하는 표준 쉘.
- **Goal**: Full-height flex 레이아웃 자동 제어 및 반응형 대응 내장.

## 2. Data Grid (Basic Table)
고밀도 인터랙티브 테이블 컴포넌트.
- **Goal**: 행 선택(Selection), 가변 높이(Density), Sticky Header 지원.

## 3. ResizablePanel
사용자가 패널 너비를 조정할 수 있게 하는 레이아웃 프리미티브.
- **Goal**: 드래그 핸들 및 최소/최대 너비 제약 설정 지원.

## 4. Overlay System (Preview)
공통 `<Overlay>` 프리미티브를 공유하는 고수준 오버레이.
- **Goal**: 포커스 트래핑, Esc 닫기, 표준 진입 애니메이션 적용.

## 5. Search & Filter (Preview)
전역 검색 및 필터링을 위한 컴포넌트 (`SearchFilterBar`).

## 6. Empty State
데이터가 없는 상태를 위한 표준 일러스트 및 가이드 컴포넌트.
