# Radius System

Border Radius 토큰은 UI 요소의 부드러움을 결정합니다.
일관된 곡률을 사용하여 유기적인 느낌을 전달합니다.

## Tokens

*   **none (0px)**: 직각, 전체 화면 컨테이너 등
*   **sm (4px)**: 작은 버튼, 입력 필드, 체크박스
*   **md (8px)**: 카드, 대형 버튼, 썸네일
*   **lg (12px)**: 모달, 대형 패널
*   **xl (16px)**: 플로팅 아일랜드, 알림 토스트
*   **full (9999px)**: 캡슐(Capsule) 형태의 버튼, 태그, 아바타
