## Usage Examples

### Basic Layer Component

```tsx
import { Layer } from '@/components/ui/Layer';

// Layer 2 - Base Surface (Sidebar)
<Layer level={2} rounded="lg" className="p-4">
  <h2>Sidebar Content</h2>
</Layer>

// Layer 3 - Primary Surface (Editor)
<Layer level={3} rounded="lg" className="p-6">
  <CodeEditor />
</Layer>

// Layer 5 - Floating (Dropdown)
<Layer level={5} rounded="md" className="p-2">
  <DropdownMenu />
</Layer>
```

### Custom Background Color

```tsx
// Use warm variant for navigation
<Layer level={2} rounded="md" className="bg-layer-2-warm">
  <WorkspaceNav />
</Layer>

// Use cool variant for content panels
<Layer level={2} rounded="md" className="bg-layer-2-cool">
  <FileTree />
</Layer>
```

### Stacking Layers

```tsx
<Layer level={0} className="app-container">
  {/* Background */}

  <Layer level={2} className="sidebar">
    {/* Sidebar */}

    <Layer level={3} className="editor">
      {/* Editor */}

      <Layer level={5} className="dropdown">
        {/* Dropdown appears on top */}
      </Layer>
    </Layer>
  </Layer>

  <Layer level={6} className="modal">
    {/* Modal overlays everything */}
  </Layer>
</Layer>
```

## Design System Integration

### TailwindCSS Configuration

The layer system is configured in `tailwind.config.js`:

```js
theme: {
  extend: {
    colors: {
      'layer-0': 'rgb(var(--layer-0) / <alpha-value>)',
      'layer-1': 'rgb(var(--layer-1) / <alpha-value>)',
      'layer-2': 'rgb(var(--layer-2) / <alpha-value>)',
      // ... layer-3 to layer-6
    },
    boxShadow: {
      'layer-0': 'var(--shadow-layer-0)',
      'layer-1': 'var(--shadow-layer-1)',
      // ... layer-2 to layer-6
    }
  }
}
```

### CSS Variables

All layer tokens are defined as CSS custom properties in `src/styles/design-tokens.css`:

```css
:root[data-theme="light"] {
  --layer-0: 246 246 246; /* #f6f6f6 */
  --layer-1: 242 242 242; /* #f2f2f2 */
  --layer-2: 255 255 255; /* #ffffff */
  /* ... */
}

:root[data-theme="dark"] {
  --layer-0: 28 28 28;    /* #1c1c1c */
  --layer-1: 24 24 24;    /* #181818 */
  --layer-2: 43 43 43;    /* #2b2b2b */
  /* ... */
}
```
