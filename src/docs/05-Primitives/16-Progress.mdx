# Progress

Visualizes what percentage of a task is complete.

<div style={{ display: 'flex', flexDirection: 'column', gap: '24px', padding: '24px', background: 'var(--surface-base-bg)', borderRadius: '12px', border: '1px solid var(--border-subtle)', maxWidth: '400px' }}>
    <Progress value={33} />
    <Progress value={66} intent="success" />
    <Progress value={80} intent="warning" size="sm" />
</div>

## Usage

```tsx
import { Progress } from '@/components/ui/Progress';

<Progress value={50} intent="primary" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `number` | - | Current value. |
| `max` | `number` | `100` | Maximum value. |
| `intent` | `primary` \| `success` \| `warning` \| `critical` \| `neutral` | `primary` | Color intent. |
| `size` | `sm` \| `md` \| `lg` | `md` | Height of the progress bar. |
